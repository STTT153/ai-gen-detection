# Dataset

## Introduction
Our dataset contains 6 categories:

1. **Pure real**: The entire image is created by a human painter.
2. **Pure AI**: The entire image is generated by an AI model.
3. **Real on AI**: The main subject (e.g., a person) is painted by a human, while the background is generated by AI.
4. **AI on real**: The background is painted by a human, while the foreground character is generated by AI.
5. **Random AI**: Randomly selected areas are generated by AI.
6. **Specific AI**: A specific area (e.g., face, clothing, or hands) is generated by AI, which is rich in sementics.

These scenarios are designed to cover all possible cases where a "cheating" artist might use AI tools to generate parts of their artwork while falsely claiming to be AI‑free.

## How these dataset is generated
1. **Pure real**: Dataset directly downloaded from Kaggle "pixiv-2020"
2. **Pure AI**: Generate by hand, use `anumageine-anything-v5.ipynb`
3. **Real on AI** and **AI on REAL**: Use `person_background_cutmix`
5. **Random AI**: Use `random-ai-substitution`
6. **Specific AI**: Use `perception_segmentation_inpaint`

Note: `inpaint_pipeline.py` is a failure attempt

## Data format
image: 512*512 .png file
mask: corresponding .npy file 1 stands for ai-gen, 2 stands for real
